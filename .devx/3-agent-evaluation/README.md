# Agent Evaluation Workshop

<img src="_static/robots/magician.png" alt="Workshop Robot Character" style="float:right;max-width:300px;margin:25px;" />

The Agent Evaluation Workshop teaches you how to measure and improve the quality of your AI agents using systematic evaluation techniques. You'll learn to evaluate both the **Report Generation Agent** and **IT Help Desk Agent** you built in previous modules using industry-standard metrics, LLM-as-a-judge approaches, and NVIDIA's evaluation tooling. Unlike manual testing, systematic evaluation provides quantitative insights into agent performance, helping you identify weaknesses and track improvements over time.

This workshop will help you understand how to evaluate AI agents in production. At the end of this workshop, you will take home:

* Understanding of key evaluation metrics for agentic systems
* Knowledge of RAGAS metrics for RAG evaluation
* Experience with LLM-as-a-judge evaluation techniques
* Hands-on implementation using NVIDIA models for evaluation
* A comprehensive evaluation framework for your agents
* Best practices for continuous agent improvement

The entire workshop can take anywhere from 1.5 to 3 hours to complete.

## Learning Objectives

By the end of this workshop, you'll know how to:
- Design evaluation datasets for different agent types
- Implement RAGAS metrics for RAG system evaluation
- Use NVIDIA models as judges for qualitative assessment
- Interpret evaluation results and identify improvement areas
