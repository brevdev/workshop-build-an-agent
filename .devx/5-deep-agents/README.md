# Deep Agents Workshop

<img src="_static/robots/magician.png" alt="Workshop Robot Character" style="float:right;max-width:300px;margin:25px;" />

In Modules 1 through 4, you built agents that reason, call tools, retrieve knowledge, and even execute shell commands. In this capstone module, you'll graduate from those shallow ReAct agents to **deep agents** — autonomous systems that can plan, code, execute, delegate, and self-manage. These are the same architectural patterns behind products like deep research assistants and AI coding agents. You'll build a production-grade deep agent using NVIDIA models, secure it with Docker sandboxing, and understand the architecture that makes it all work.

This learning module can take around 1 to 2 hours to complete.

## At the end of this module, you will take home:

- Understanding of deep agent architecture and the four pillars (planning, delegation, memory, skills)
- Knowledge of how deep agents differ from shallow ReAct agents
- Hands-on experience building a deep agent with the deepagents library and an interactive UI
- Familiarity with real-world deep agent applications (deep research, coding agents)
- Best practices for sandboxing and secure autonomous execution

## Learning Objectives

By the end of this module, you'll know how to:
- Explain what makes deep agents different from shallow ReAct agents
- Describe the four pillars of deep agents (planning, delegation, memory, skills)
- Identify real-world use cases where deep agents excel over simpler approaches
- Build a deep agent with planning, filesystem, shell, and sub-agent capabilities
- Extend agents with MCP tools and custom skills
- Implement Docker-based sandboxing for agent isolation
- Apply defense-in-depth security with HITL + sandboxing
- Deploy and monitor agents in production with LangSmith
